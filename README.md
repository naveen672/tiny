# Tiny Prism Labs - Corporate Website

A premium, modern corporate website for Tiny Prism Labs Private Limited, specializing in Embedded Systems, Edge AI, TinyML, and Industrial AI solutions.

## 🚀 Tech Stack

- **Frontend**: React 18, React Router, Framer Motion
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Backend**: Node.js, Express
- **Icons**: React Icons

## 📦 Installation

```bash
# Install dependencies
npm install
```

## 🛠️ Development

```bash
# Run development server
npm run dev

# Visit http://localhost:3000
```

## 🏗️ Build

```bash
# Create production build
npm run build

# Preview production build
npm run preview
```

## 🖥️ Production Server

```bash
# Start Node.js server (serves built React app)
npm run server
```
